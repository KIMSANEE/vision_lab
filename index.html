VISION LAB.

Versatile Intelligent Sensor IC and On-chip Systems Lab

Our group focuses on creating intelligent vision sensor circuits and systems that integrate CMOS vision sensor IC with spiking neural networks on a single-chip platform.

VISION LAB에서는 CMOS 비전 센서 IC 설계와 스파이킹 신경망을 단일 칩 플랫폼에 통합하여 시각 정보를 직접 처리하는 지능형 비전 센서 회로 및 시스템 구현을 위해 연구한다. 이를 위해 LiDAR를 포함한 고성능 비전 센서 아키텍처와 mixed-signal SNN 회로, AI 학습 알고리즘, 광검출기 등 다양한 기술 분야를 연구하여 차세대 비전 센서 패러다임을 제시한다.

We invite you to browse our website to learn more about us and our research!

Research

CMOS LiDAR Sensor

How can we measure distance?

  • Direct time-of-flight method (d-ToF) :
  • Directly measure time between laser emission and the arrival of its echo

LiDAR Sensor IC Research Topics

  • SPAD (Single-photon Avalanche diode)
    Device and process optimization with analog front-end circuit design for single-photon detection.
  • Time-to-digital converter (TDC)
    Sub-nanosecond precision TDC architectures with area efficiency and array integration capability.
  • Histogram memory
    Architectural optimization of memory systems that occupy majority of LiDAR chip area.
  • Depth calculator
    Advanced algorithms for accurate distance estimation under non-linear responses and super-resolution processing.

LiDAR sensor IC development requires comprehensive research across all aspects of electronics engineering, from device-level optimization through analog circuits, digital circuits and architectures, to post-processing algorithms!

1. VCSEL Scanning High-Resolution LiDAR Sensor with Area-Efficient SPAD AFE and Histogram Memory (ISSCC 2024)

This chip presents a high-resolution solid-state LiDAR sensor featuring a 320×240 pixel array with innovative area-efficient design approaches.

The sensor implements a compact 6-transistor nMOS-only SPAD analog front-end (AFE) circuit with column-shared active recharging, achieving 5ns dead time while minimizing pixel area.

A novel priority histogram memory architecture reduces memory requirements to 1/16, selectively storing only high-probability echo candidates.

The 2-step histogramming TDC enables efficient 2-echo detection with reconfigurable memory allocation for both coarse and fine measurements.

Key Contributions:
  • Area-efficient 6-transistor SPAD pixel design
  • Significant memory reduction through priority histogram memory
  • 98.6% detection success rate under 100klx ambient light
  • 5.7cm depth accuracy with 0.02% precision at 48m range

2. CMOS LiDAR Sensor with Enhanced Noise Immunity (S.VLSI 2021 / IEEE Sensors 2022)

This chip addresses critical challenges in LiDAR systems operating under harsh conditions with high background noise and inter-LiDAR interference.

The sensor implements a multi-event weighted histogramming TDC with 4×4 macro-pixels and an area-efficient 11-bit two-step TDC architecture using mixed-signal accumulators.

A novel pre-post weighted histogram filter selectively detects time-correlated signals while rejecting background noise from sunlight and dark counts.

Additionally, an infinite interference canceling (IIC) scheme with pulse-position and time-difference modulation enables robust operation in multi-LiDAR environments using on-chip SPAD-based random number generation.

Key Contributions:
  • Area-efficient histogramming architecture with enhanced dynamic range
  • Effective background noise and interference mitigation
  • 8.08-cm accuracy at 32m range under extreme conditions (105-klx sunlight)


3. 36-Channel Mechanical Scanning LiDAR Sensor with On-chip SPAD Array (S.VLSI 2020 / JSSC 2021) - Korea's 1st LiDAR Sensor SoC

This chip features a compact scanning LiDAR sensor with a 36-channel SPAD array and area-efficient 11-bit in-situ histogramming TDC (3000 × 78 µm² per channel).

The system incorporates an embedded interference filter enabling reliable operation even with interference from 32 different LiDAR sensors.

Combined with dual laser diodes and hybrid mirror beam scanner, the sensor achieves high-resolution imaging (2200 × 36) with wide field-of-view (120° × 8°).

Key Contributions:
  • Area-efficient mixed-signal all-event histogramming TDC architecture
  • Robust interference filtering for multi-LiDAR environments
